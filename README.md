# 💊 Drug Store Management System

## 📝 Project Overview
This is a Java-based desktop application built using **Java Swing** for managing a drug store. The system provides an easy-to-use interface for handling essential operations such as user authentication and managing drug records stored in a database.

---

## 🎯 Project Purpose
The main goal of the project is to simplify the workflow in a drug store by digitizing and automating operations such as registering users, managing drug data, and performing complex queries.

---

## ⚙️ Key Features

### 🔐 Authentication
- **Login and Registration** for users

### 💊 Drug Management
- **Add**, **Update**, **Search**, and **Delete** drug information
- View **all drugs** in the database
- Display **total number of drugs** stored

### 🧠 Query Functions
- Convert **drug names to uppercase** using SQL functions
- **Join** between `users` and `drugs` table using `user_id`
- Use of **subqueries** and **UNION** to retrieve combined and filtered results

### 🏠 Navigation
- Home Page with navigation options:
  - Login Page
  - Registration Page
  - Drug Management Main Page

---

## 🛠️ Technologies Used
- **Java Swing** (GUI)
- **JDBC** (Java Database Connectivity)
- **MySQL** or **SQLite** (for backend database)
- **SQL Queries** (JOIN, SUBQUERIES, UNION, UPPER, COUNT)
